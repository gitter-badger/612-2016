Constructor & Destructor Documentation

VictorSP::VictorSP ( uint32_t  channel)
explicit

Note that the VictorSP uses the following bounds for PWM values. These values should work reasonably well for most controllers, but if users experience issues such as asymmetric behavior around the deadband or inability to saturate the controller in either direction, calibration is recommended. The calibration procedure can be found in the VictorSP User Manual available from Vex.

2.004ms = full "forward" 1.52ms = the "high end" of the deadband range 1.50ms = center of the deadband range (off) 1.48ms = the "low end" of the deadband range 0.997ms = full "reverse" Constructor for a VictorSP

Parameters
channelThe PWM channel that the VictorSP is attached to. 0-9 are on-board, 10-19 are on the MXP port
